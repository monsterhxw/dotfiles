# Tmux Configuration
## Theme

My personal tmux configuration with Nord-inspired theme.

Theme file: [theme.conf](./themes/theme.conf)

Load the theme in **tmux.conf**
```conf
source-file "$YOUR_PATH/theme.conf"
```
